Booking Server and Client - REST Server

Develop website system for reservation process


The project aims to make the work in the Tourism easer. The client part of the system is providing various opportunities for users and agencies. If the user is an agent, besides searching and making reservations, they can also create new hotels, add their descriptions, photos and set prices, etc. 

The server store the information in real-time and can send this information to other systems. The system allows bookings in real-time.

Here are some of the main feature that the project included:

  • Real-time reservations
  • Modern design
  • REST Server - ease to implement
  • Set hotel description and prices

Technologies involved:
  • JavaEE
  • Spring Framework
  • Spring Boot
  • Spring Security
  • Hibernate ORM
  • Thymleaf
  • HTML, CSS, JS, Bootstrap
  • MySQL
  • Apache Maven
  • WWW.SINEVA.BG
