Desktop application for work management

Terra

The program main responsibility was to give the clarity of what had been manufactured and what each of the workers had done on their own. The terra desktop application for managing the workflow is architecture from two smaller programs that communicates with each other via the internet.  

The first was the manager program. This program manager of the factory can set different details that can be manufactured and reported by the workers. The manager can generate reports for the work that was done. This helps him to make decisions. Also, the program keeps a complete history of the work progress.

The second program gives the employee the ability to report their work. The design was user-friendly so that the people who work with it to not lose time at writing the report.

Technologies involved:
  • JAVA
  • MS SQL
  • SWING
